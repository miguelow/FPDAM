Tablas en HTML

Un tabla es un conjunto de celdas donde podemos alogar distintos contenidos: imagenes, texto, formulario, vídeos, etc.
Las tablas en HTML se definen a través de tres etiquetas que son su base principal:
  • table:con esta etiqueta definimos la tabla en HTML
  • tr:es aquella que divide la tabla en filas
  • td:es aquella que divide la fila en celdas

Las tablas, fueron los elementos principales de maquetación en HTML4.
Con la llegada de la normativa de HTML5, se empezó a cambiar las tablas por etiquetas semánticas y por
divs para poder maquetas y dar una mayor libertad a nuestros diseños y elementos de la web.

Propiedades o Elementos de una Tabla

  • table: tiene etiqueta de apertura y cierre. Define una tabla en HTML
  • caption: tiene etiqueta de apertura y cierre. Se utiliza para poner un título a la tabla
  • Atributo align=top/bottom: permite colocar el título arriba o abajo de nuestra tabla
  • tr: tiene etiqueta de apertura y cierre. definir la fila de una tabla
  • td: tiene etiqueta de apertura y cierre. define la celda (columna) de una tabla
  • thead: Define la cabecera de una tabla.
  • th: Define una celda de tipo cabecera en la tabla.
  • tbody: define el contenido de un tabla
  • tfoot: define un pie de la tabla

Combinación de tablas en HTML

Con las etiquetas td y th, podemos utilizar lo que se conoce como combinación de celdas en tablas.
La combinación de celdas en tablas, no es más que "unificar" celdas de manera que podamos
diseñar distintas estructuras y formas dentro de nuestra tabla con el fin de hacer más
atractivas visualmente y colocar mejor el contenido de las mismas.
Los atributos que utilizaremos para ellos son los siguientes:
  • colspan: indicar el númoer de filas que ocupa la celda (combinación de filas)
  • rowspan:indica el número de columnas que icula la celda (combinación de columnas)
