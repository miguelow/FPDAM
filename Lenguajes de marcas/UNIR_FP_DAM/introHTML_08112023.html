Separador de secciones mediante la etiqueta HR

La etiqueta hr, es un elemento del que disponemos en HTML para poder hacer separaciones
de secciones o divisiones de texto, con el fin de poder tener todo mucho más
organizado Ejemplo de línea hr. Se utiliza la monoetiqueta hr.
Si utilizamos los atributos width, size y color, podemos establecer lo que son
tamaño de la línea (en %), grosor de la línea (px) y color.
Etiqueta Preformateado (pre)

La etiqueta pre, con apertura y cierre, es un tipo de tag que nos permite
insertar texto de manera que se visualice en nuestra página web tal y como
la estamos escribiendo. Es decir, de una manera preformateada.
La etiqueta code permite insertar código en un documento HTML.
Siempre se suele usar junto con la etiqueta pre.
