Imágenes en HTML

En informática podemos distinguir dos tipos de imágenes:
  • Imágenes de Mapa de bits: formadas por puntos de colores.
    Ejemplo: imagenes de 8bits (videojuegos antiguos)
  • Imágenes vectoriales: formadas por formas geométricas. Eje: ilustraciones, imágenes de carteles, etc.

Ventajas y Desventajas Imágenes Vectoriales (SVG)

  • Ventaja:Ocupan poco espacio y se pueden ampliar a cualquier escala sin que pierda calidad de imagen
  • Inconveniente: No permiten representar imágenes sin formas definidas. Eje: retrato.

Ventajas y Desventajas Imágenes Mapa de Bits (bpm)

  • Ventaja:Si que permiten representar imáganes sin formas definidas
  • Inconveniente: Pierden mucha calidad al ampliardas y ocupan mucho espacio en el disco duro.

Etiqueta img para insertar fotos en HTML

Para insertar fotos en HTML, utilizaremos la etiqueta img.
La estructura que tiene es la siguiente:
  • Monoetiqueta img (no tiene etiqueta de cierre).
  • Atributo src (source). Sirve para poder vincular la foto que queremos mostras.
  • La dirección donde se ubica la foto en donde tenemos que indicar a parte del nombre,
    el formato (extensión) que tiene. Ejemplo: foto.png /.jpg /.gif /.svg...

Ejemplo de foto insertada con etiqueta img.

Atributos asociados a las fotos en HTML

  • src: indica la dirección (URI) de acceso a la imagen
  • alt: para indicar un breve texto en caso de que la imagen falle.
  • longdesc: indica es una URI de una descripción de una foto o imagen en caso de tener
    asociado un mapa de imágenes, describir imágenes.
  • height y width: para dar altura y anchura a la foto
  • usemap: Sirve para asociar a la imagen un mapa de imágenes (como un marcador)
  • ismap: sirve para definir un mapa de imágenes
  • title: a forma de tip, muestra información en un cuadradito amarillo de la imagen cuando
    pasamos el ratón por encima.

Figure en HTML

Permite asociar un texto con una imagen.
  • figure: es la etiqueta padre. Es decir, el resto de etiquetas dependerán o caerán de esta
  • img: es una etiqueta para insertar imágenes
  • figcaption: es una etiqueta que contendrá el texto asociado a la imagen
